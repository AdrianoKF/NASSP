Panel controls and displays reference

This section is a reference of each working switch and each display on the Project Apollo panels. It's intended mainly as a developer reference but of course it's also useful for users just in case they are curious about all these switches.

Controls and displays are presented in form of a tabulated list in numerical order by panel and area number. These tables are strongly related to the Apollo Operations Handbook, see below for more details. Each spacecraft has it's own reference table. The following is a detailed explanation of the columnar data presented in the tabulated lists:

Name

Given the exact nomenclature of a particular control or display. The name of the switch or control in the C++ code should be similar like shown in the following example: The name of the CABIN FAN switches in the code are CabinFan1Switch and CabinFan2Switch.
Panel Locator The panel locator provides the "coordinates" of the display or control. Please click here (or on the image to the right) to download the related figures from the Apollo Operations Handbook. Firstly there is the panel number, larger panels are further divided into areas. In most instances, the area correlates to a specific spacecraft system. Panels 1 through 16 have grid reference numerical marks horizontally and alphabetic marks vertically on the drawing.
Function Describes the function of the switch or display.  
Remarks Any useful comment. Please notice especially differences to the original function, bugs or other issues.

Indicates one of the following conditions in the respective fuel cell:

  • H2 flow rate above 0.171 lb/hr
  • O2 flow rate above 1.276 lb/hr
  • Skin temperature below 360 °F or above 475 °F
  • Condenser exhaust temperature below 150 °F or above 175 °F
  • Radiator outlet temperature below - 30 °F

HIGH GAIN ANTENNA PITCH POSITION switch
2
R
N-39

TEMPORARY TEST SWITCH FOR ECS!

If the switch is set to +90° (up) there's no additional ECS heating, each position adds 800 W of heating power, so in the -90° position (down) the coolant loop is heated with a heating power of 4800 W.

To test and play with the primary water-glycol coolant loop this switch can be used to heat the coolant loop with a boiler.

This function will be removed and replaced with the correct one in later versions.
